# Final-Paper-on-Financial-Data-Analysis-and-Visualization
金融数据分析与可视化期末论文相关代码
仓库里包含股票基本数据（30个股票基本信息.csv），年报各类词数情况以及语调指数（年报语调指标.csv），股吧评论各类词数情况以及情绪指数（评论情绪分析结果.csv）等基本数据，以及研究使用的评论情绪，年报语调和股票数据匹配对应的数据集（研究窗口股票数据.csv）。
此外还有研究使用的停用词表以及金融词典。
研究以及可视化的代码可以在与“研究窗口股票数据.csv”同一个目录的情况下运行，但是数据获取和清洗代码不可运行（代码里的部分中间文件没上传到仓库且文件路径不一致）。
